Dev PX4¶

  • https://docs.px4.io/main/zh/dev_setup/getting_started.html

Download¶

git clone https://github.com/PX4/PX4-Autopilot.git --recursive

First Build (Using the jMAVSim Simulator)¶

make px4_sitl jmavsim

pxh> commander takeoff

pxh> commander land
